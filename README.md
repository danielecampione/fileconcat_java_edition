# FileConcat – Java Edition (JavaFX GUI)

## 🇮🇹 Italiano

**FileConcat – Java Edition (JavaFX)** è una piccola applicazione desktop che permette di **unire più file di testo** in un unico file finale.  
Questa versione utilizza **Java 8+** e una semplice interfaccia grafica basata su **JavaFX**.

### Funzionalità
- Selezione multipla dei file tramite finestra di dialogo
- Riordinamento dei file prima della concatenazione
- Log delle operazioni nella finestra principale
- Salvataggio del file finale in un percorso scelto dall’utente
- Nessuna dipendenza esterna, funziona su Windows / macOS / Linux

### Requisiti
- Java 8 o superiore

### Utilizzo
1. Avvia l’applicazione (`java -jar FileConcatJavaFX.jar`)
2. Clicca **Aggiungi file**
3. Riordina i file se necessario
4. Clicca **Concatena**
5. Scegli il file di destinazione

## 🇩🇪 Deutsch

**FileConcat – Java Edition (JavaFX)** ist ein kleines Desktop‑Programm, mit dem du **mehrere Textdateien** zu einer einzigen Ausgabedatei zusammenführen kannst.  
Diese Version verwendet **Java 8+** und eine einfache **JavaFX‑Oberfläche**.

### Funktionen
- Mehrfachauswahl von Dateien über einen Dialog
- Umsortieren der Dateien vor dem Zusammenführen
- Protokollanzeige der Aktionen im Hauptfenster
- Speichern der Ausgabedatei in einem frei wählbaren Pfad
- Keine externen Abhängigkeiten, läuft unter Windows / macOS / Linux

### Voraussetzungen
- Java 8 oder höher

### Verwendung
1. Anwendung starten (`java -jar FileConcatJavaFX.jar`)
2. Auf **Dateien hinzufügen** klicken
3. Dateien bei Bedarf umsortieren
4. Auf **Zusammenführen** klicken
5. Ausgabedatei auswählen

## 🇨🇳 中文

**FileConcat – Java 版（JavaFX）** 是一个轻量级桌面应用程序，用于将**多个文本文件合并**为一个输出文件。  
本版本基于 **Java 8+**，并提供简洁的 **JavaFX 图形界面**。

### 功能
- 通过对话框一次选择多个文件
- 在合并前重新排序文件
- 在主窗口中显示操作日志信息
- 将结果保存到用户指定的路径
- 无需额外依赖，可在 Windows / macOS / Linux 上运行

### 需求
- Java 8 或更高版本

### 使用方法
1. 启动程序（`java -jar FileConcatJavaFX.jar`）
2. 点击 **添加文件**
3. 如有需要，重新排序文件
4. 点击 **合并**
5. 选择输出文件路径
